# 🚀 Project Hail Mary Data Hub

A professional Streamlit web application designed to analyze and visualize the performance of Sci-Fi movies, with a special focus on **Project Hail Mary**. This project was developed as part of the **Arts and Big Data (SOA2005-01)** course at Sungkyunkwan University.

## 📝 Description
The **Project Hail Mary Data Hub** is an interactive dashboard that collects, organizes, and visualizes financial and thematic data related to science fiction cinema. It serves as a digital portfolio that demonstrates the intersection of film media and data science.

## ✨ Key Features
*   **Movie Gallery**: Browse a curated collection of Sci-Fi films including Project Hail Mary, The Martian, and Interstellar.
*   **ROI Analysis**: Automatically calculate and visualize the Return on Investment (ROI) for various film projects.
*   **Interactive Charts**: Explore budget vs. revenue trends and genre distributions using dynamic Plotly charts.
*   **Data Management**: Add new movies to the database through a user-friendly sidebar interface.

## 🛠️ Technology Stack
*   **Language**: Python
*   **Framework**: Streamlit
*   **Visualization**: Plotly
*   **Data Handling**: Pandas

## 🚀 How to Run
To run this application locally, follow these steps:

1.  **Install Requirements**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Launch the App**:
    ```bash
    streamlit run app.py
    ```

## 📂 Project Structure
*   `app.py`: The main application code.
*   `requirements.txt`: List of necessary Python libraries.
*   `README.md`: Project documentation.
